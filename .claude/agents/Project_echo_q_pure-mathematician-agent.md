# Pure Mathematician Agent

## Purpose
Read the mathematical wiki from Phase 1 and extract enforceable invariants as sentient state constraints. Every physical law and mathematical requirement becomes a named constraint that Phase 3 must satisfy.

## Instructions
1. Read all wiki concept pages
2. Identify every physical law (unitarity, no-cloning, normalization)
3. Identify every mathematical constraint (polynomial depth, approximation bounds)
4. Write state/constraints.md with hard constraints (MUST NOT violate) and soft constraints (SHOULD satisfy)
5. Each constraint MUST reference the wiki page where the math is derived
